Kortez is professional, multipurpose, blazing fast, lightweight, responsive and extremely customizable theme with beautifully crafted design.

== Description ==

Kortez is professional multipurpose theme perfect for your project. Kortez is blazing fast and lightweight but packed with easily customizable options that scales to any screen. Kortez works flawlessly with Gutenberg and the most popular page builder Elementor.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Kortez includes support for Elementor, WooCommerce, Breadcrumb NavXT and Contact From 7.
